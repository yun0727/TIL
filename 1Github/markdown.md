# 마크다운 문법 내용 정리

## Heading

### 문서의 제목이나 소제목

1. h1 ~ h6까지 표현 가능
2. 문서의 구조를 위한 작성
3. 단순 글자 크기 조절을 위한 사용 안됨

## List

1. 목록
2. ol: 순서가 있는 리스트
3. ul: 순서가 없는 리스트

## Fenced Code Block

1. backtick 기호 3개 활용하여 작성

```python
print("hello")
```

## Inline Code Block

1. `기호 1개 활용하여 작성

마크다운은 `마크업 언어` 입니다.

## link

1. [문자열] (url)을 통해 작성

[구글](https://google.com)
